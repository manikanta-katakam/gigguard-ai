# GigGuard AI – AI-Based Income Protection for Gig Workers

👥 Team: Code Smashers

## Inspiration
Gig workers like delivery partners and drivers lose income during rain, heat, or pollution. We built GigGuard AI to protect their income using AI.

## What it does
GigGuard AI:
- Detects weather risks
- Sends alerts to workers
- Provides insurance protection
- Allows monthly plan selection
- Includes AI chatbot support
- Admin monitors workers and claims

## How we built it
- Designed using Figma prototype
- Frontend: React (concept)
- Backend: Node.js (concept)
- Database: MongoDB
- AI: Weather-based risk detection
